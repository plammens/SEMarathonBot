info - General information about this bot and credits
start - Awaken bot and start listening for commands
new_marathon - Create new marathon
settings - View current settings for marathon
add_participants - Add participants to the marathon
set_sites - Set the SE sites to be tracked during the marathon
set_duration - Set the duration for the marathon
start_marathon - Start the marathon!
leaderboard - Show marathon leaderboard
status - Show running status
pause_marathon - Pause an ongoing marathon
stop_marathon - Stop an ongoing marathon prematurely
cancel - Cancel an ongoing operation
reset - Forget & reset the current conversation
